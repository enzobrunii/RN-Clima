### RN-Clima

##### APIS
Se utiliza la api del clima de [Wunderground]

Se debe ingresar el `API KEY` en los siguientes Archivos

`API_KEY = '***TU***API***KEY***'`

/src/services
 - forecast.js
 - weather.js
